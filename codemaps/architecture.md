## Key Modules

### Orchestration Core
- **FastAPI Server**: Handles HTTP requests, initiates tasks, and provides health checks.
- **NATS Worker**: Background process subscribing to `tasks.submit`, executing agents, and archiving results.

### Messaging & Persistence
- **NATS JetStream**: Durable message delivery for tasks and a KV store for ephemeral result caching.
- **SQLite**: Permanent record of all task lifecycles, metadata, and final results.

### Security Layer
- **Master Secret**: PBKDF2-derived key for encrypting the local keystore.
- **Encryption**: AES-GCM for protecting sensitive API keys at rest.

## Data Flow
[User/Client] -> [NexusSDK] -> [FastAPI API] -> [NATS Bus] -> [NexusWorker] -> [Agent/Tools] -> [NATS KV/DB] -> [User/Client]
